Hello!

This Web app allows you to select a land of your choice from a given area of your choice with the best user interface. Also if you have already selected any land, you can you can view the pros and cons in there. Our app is embedded with different layers of maps that provides data about the nearby features such as schools, bus stops, railways station….. as well as the soil maps ,ground water prospects and other such maps that provides you the data about any particular place.

Buttons

To select the area in which you want search for the land.
After selecting the requied features click the done button

To select the types of features to view nearby a point by cliking the marker at a point.

To clear markers and to return to Gmaps view

Overlay Dropdown

This allows you to overlay the differnt layers of map that are embedded in database.
You can turn them on and off.

All the map features are obtained from Open Street Maps and the algorithim is based on the features provided by OSM.
